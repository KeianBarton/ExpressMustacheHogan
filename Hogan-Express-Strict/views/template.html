Lambdas example

Non-false value example

When a value is not a list and non-false, it will be used as context for any further nested values

{{#nonFalse}} Nested value is: {{nestedVal}} {{/nonFalse}}

Inverted sections

{{#invertedSection}}

This will be rendered if the key exists, is true, or is a non-empty list

{{/invertedSection}} {{^invertedSection}}

This will be rendered if the key doesn't exist, is false, or is an empty list

{{/invertedSection}}
